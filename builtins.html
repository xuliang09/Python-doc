builtins函数手册

input

语法
    input([prompt])

参数
    prompt：提示用户的字符串

功能
    接受标准输入流的字符串

例子
    >>> a = input('input a integer\n')
    >>> print(int(a))
    input a integer
    2
    2

map

语法
    map(func, iter)

参数
    func：应用到iter上的函数
    iter：可迭代的对象

功能
    对iter中的每个元素应用func

例子
    >>> s = ['1', '2', '3']
    >>> map(int, s)
    < map object at 0x0000025D905E94A8 >
    >>> list(map(int, s))
    [1, 2, 3]

while和for

功能
    写一下while和for后面接else的情况，一句话：只要不执行break就运行else

例子
    i=1
    while i < 10:
        i+=1
        # break，这个地方如果执行了break，else里面的内容就不会执行
    else:
        print('i am here')
    OUT[1]: i am here
