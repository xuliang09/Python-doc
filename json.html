json函数手册

dumps

语法
    json.dumps(obj)

功能
    将一个对象转化成一个字符串

参数
    obj：符合json格式的对象

例子
    >>> data = [ { 'a' : 1, 'b' : 2, 'c' : 3, 'd' : 4, 'e' : 5 } ]
    >>> data
    [{'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5}]
    >>> import json
    >>> json.dumps(data)
    '[{"a": 1, "b": 2, "c": 3, "d": 4, "e": 5}]

dump

语法
    json.dump(obj, fp)

功能
    将一个对象序列化并存储到文件中

参数
    obj：符合json格式的对象
    fp：文件的句柄

例子
    >>> data = [ { 'a' : 1, 'b' : 2, 'c' : 3, 'd' : 4, 'e' : 5 } ]
    >>> data
    [{'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5}]
    >>> json.dump(data, open('a.txt', 'w'))

loads

语法
    json.loads(obj)

功能
    将字符串转化成一个json对象

参数
    obj：符合json格式的对象

例子
    >>> s
    '[{"a": 1, "b": 2, "c": 3, "d": 4, "e": 5}]'
    >>> ds = json.loads(s)
    >>> ds
    [{'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5}]

loads

语法
    json.load(fp)

功能
    从文件中读取一个json对象

参数
    fp：文件的句柄

例子
    >>> ds = json.load(open('a.txt'))
    >>> ds
    [{'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5}]
