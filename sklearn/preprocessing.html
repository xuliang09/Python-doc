sklearn.preprocessing函数手册

scale

语法
    sklearn.preprocessing.scale(X, axis=0, with_mean=True, with_std=True, copy=True)

功能
    Standardize a dataset along any axis, Center to the mean and component wise scale to unit variance

参数
    X：进行scale的数据
    axis：取0或1。取0表示对于每个feature进行归一化，取1表示对每个sample进行归一化
    with_mean：
    with_std：
    copy：

例子
    >>> from sklearn import preprocessing
    >>> import numpy as np
    >>> X = np.array([[ 1., -1., 2.],
    ... [ 2., 0., 0.],
    ... [ 0., 1., -1.]])
    >>> X_scaled = preprocessing.scale(X)
    >>> X_scaled
    array([[ 0. ..., -1.22..., 1.33...],
     [ 1.22..., 0. ..., -0.26...],
     [-1.22..., 1.22..., -1.06...]])

StandardScaler

语法
    sklearn.preprocessing.StandardScaler(copy=True, with_mean=True, with_std=True)

功能
    与函数sklearn.preprocessing.scale功能一样，但其生成了一个对象，可以重复使用

参数
    copy：默认True。False表示直接改变原数据里面的内容
    with_mean：
    with_std：

例子
    >>> from sklearn.preprocessing import StandardScaler
    >>> data = [[0, 0], [0, 0], [1, 1], [1, 1]]
    >>> scaler = StandardScaler()
    >>> print(scaler.fit(data))
    StandardScaler(copy=True, with_mean=True, with_std=True)
    >>> print(scaler.transform(data))
    [[-1. -1.]
    [-1. -1.]
    [ 1. 1.]
    [ 1. 1.]]
    >>> print(scaler.transform([[2, 2]]))
    [[ 3. 3.]]

MinMaxScaler

语法
    sklearn.preprocessing.StandardScaler(feature_range=(0, 1), copy=True)

功能
    可以将数据归一化到指定范围内

参数
    feature_range：默认(0, 1)。归一化后每个数据值的范围
    copy：默认True。False表示直接改变原数据里面的内容

例子
    >>> scaler = preprocessing.StandardScaler().fit(X)
    >>> scaler
    StandardScaler(copy=True, with_mean=True, with_std=True)
    >>> scaler.mean_
    array([ 1. ..., 0. ..., 0.33...])
    >>> scaler.transform(X)
    array([[ 0. ..., -1.22..., 1.33...],
     [ 1.22..., 0. ..., -0.26...],
     [-1.22..., 1.22..., -1.06...]])
    >>>#可以直接使用训练集对测试集数据进行转换
    >>> scaler.transform([[-1., 1., 0.]])
    array([[-2.44..., 1.22..., -0.26...]])

minmax_scale

语法
    sklearn.preprocessing.minmax_scale(X, feature_range=(0, 1), axis=0, copy=True)

功能
    可以将数据归一化到指定范围内

参数
    X：进行scale的数据
    feature_range：默认(0, 1)。归一化后每个数据值的范围
    copy：默认True。False表示直接改变原数据里面的内容
    axis：取0或1。取0表示对于每个feature进行归一化，取1表示对每个sample进行归一化

例子
    >>> from sklearn.preprocessing import minmax_scale
    >>> x = [0,1,2,3,4,5]
    >>> minmax_scale(x)
    array([0. , 0.2, 0.4, 0.6, 0.8, 1. ])
    >>> y = [[0,0,0],[1,1,1],[2,2,2]]
    >>> minmax_scale(y)
    array([[0. , 0. , 0. ],
     [0.5, 0.5, 0.5],
     [1. , 1. , 1. ]])
