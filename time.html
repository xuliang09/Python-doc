time函数手册

time

语法
    time.time()

功能
    一般用来计算程序运行的时间

例子
    import time
    time_b = time.time()
    func()
    time_a = time.time()
    print(time_a - time_b)

clock

语法
    time.clock()

功能
    一般用来计算程序运行的时间，用的比time更多

例子
    import time
    clock_b = time.clock()
    func()
    clock_a = time.clock()
    print(clock_a - clock_b)

strptime

语法
    time.strptime(string, format)

功能
    将字符串形式的时间转化成struct_time的时间表示

参数
    string：字符串形式的时间
    format：格式字符串

例子
    >>> import time
    >>> print(time.strptime('2018-07-02 13:07:54', '%Y-%m-%d %H:%M:%S'))
    time.struct_time(tm_year=2018, tm_mon=7, tm_mday=2, tm_hour=13, tm_min=7, tm_sec=54, tm_wday=0, tm_yday=183, tm_isdst=-1)

sleep

语法
    time.sleep(seconds)

功能
    进程睡眠seconds秒

参数
    seconds：进程睡眠的时间

例子
    import time
    time.sleep(3)
