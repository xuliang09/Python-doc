matplotlib函数手册

scatter

语法
    scatter(x, y, s=None, c=None, marker=None, cmap=None, norm=None, vmin=None, vmax=None, alpha=None, linewidths=None, verts=None, edgecolors=None, hold=None, data=None, **kwargs)

功能
    画散点图，便于查看数据的分类效果

参数
    x, y：待显示的数据
    s：size，数据点的显示大小
    c：color，数据点的显示颜色

例子
    >>> import matplotlib
    >>> import matplotlib.pyplot as plt
    >>> fig = plt.figure()
    >>> ax = fig.add_subplot(111)
    >>> ax.scatter([0, 1], [0, 1], s=15, c='r')
    <matplotlib.collections.PathCollection object at 0x00000154950A5DA0>
    >>> plt.show()

plot

语法
    plot(*args, **kwargs)

功能
    画折线图

参数
    label：给曲线取个名字

例子
    >>> from matplotlib import pyplot as plt
    >>> x = [500, 1000, 1500, 2000, 2500, 3000]
    >>> y1 = [4, 10, 17, 24, 45, 60]
    >>> y2 = [11, 31, 64, 105, 170, 240]
    >>> plt.xlabel('Number of categories')
    >>> plt.ylabel('Seconds for one iteration')
    >>> plt.plot(x, y2, 'ys-', label="CC-BiBTM")
    >>> plt.plot(x, y1, 'go-', label="Fast CC-BiBTM")
    >>> plt.legend()
    >>> plt.show()
