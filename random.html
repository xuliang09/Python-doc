random函数手册

randint

语法
    random.randint(a, b)

功能
    返回一个属于区间[a, b]的整数, 包括a和b

参数
    a, b：两个整数

例子
    >>> import random
    >>> random.randint(1, 3)
    2
    >>> random.randint(1, 3)
    3
    >>> random.randint(1, 3)
    1

random

语法
    random.random()

功能
    返回一个属于区间[0, 1)的浮点数

例子
    >>> import random
    >>> random.random()
    0.81332125135732

seed

语法
    random.seed(a=None, version=2)

功能
    设置一颗种子，同一颗种子下的随机数是一致的

参数
    a：种子数

例子
    >>> import random
    >>> random.seed(2)
    >>> random.random()
    0.9560342718892494
    >>> random.seed(2)
    >>> random.random()
    0.9560342718892494
