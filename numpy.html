numpy函数手册

tile

语法
    tile(A, reps)

功能
    重复A，次数为reps

参数
    A：
    reps：

例子
    >>> a = np.array([0, 1, 2])
    >>> np.tile(a, 2)
    array([0, 1, 2, 0, 1, 2])
    >>> np.tile(a, (2, 2))
    array([[0, 1, 2, 0, 1, 2],
    [0, 1, 2, 0, 1, 2]])
    >>> np.tile(a, (2, 1, 2))
    array([[[0, 1, 2, 0, 1, 2]],
    [[0, 1, 2, 0, 1, 2]]])

zeros

语法
    zeros(shape, dtype=float, order='C')

功能
    一个给定形状和类型的用0填充的array

参数
    shape：形状
    dtype：数据类型，可选参数，默认numpy.float64
    order：可选参数，C代表与c语言类似，行优先；F代表列优先

例子
    >>> np.zeros(5)
    array([ 0., 0., 0., 0., 0.])
    >>> np.zeros((5,), dtype=np.int)
    array([0, 0, 0, 0, 0])
    >>> np.zeros((2, 1))
    array([[ 0.], [ 0.]])
    >>> s = (2,2)
    >>> np.zeros(s)
    array([[ 0., 0.], [ 0., 0.]])
    >>> np.zeros((2,), dtype=[('x', 'i4'), ('y', 'i4')]) # custom dtype
    array([(0, 0), (0, 0)], dtype=[('x', '<i4'), ('y', '<i4')])
